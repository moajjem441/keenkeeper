# 👥 KeenKeeper

> Keep your relationships strong, intentional, and never out of touch.

KeenKeeper is a relationship tracking and communication management web application that helps you maintain meaningful connections with friends and acquaintances. It allows you to log interactions, set communication goals, and track relationship health over time.

---

## 🌐 Live Demo

🚀 Try it here:  
https://keenkeeper-gamma.vercel.app

## 🧩 Key Highlights

- 👥 Friend relationship management  
- ⚡ Quick interaction logging (call, message, video)  
- 🎯 Communication goal tracking  
- 📊 Relationship health indicators  
- 📚 Interaction history tracking  
- 📱 Fully responsive design  
- 💾 Local storage-based data system  

---

## ✨ Features

### 👥 Friend Management
Add, update, and manage friends with detailed profiles and interaction history.

### 🎯 Communication Goals
Set how often you want to communicate with each friend and track your progress.

### ⚡ Quick Check-Ins
Log interactions instantly:
- 📞 Phone calls  
- 💬 Messages  
- 🎥 Video calls  

### 📊 Relationship Status
Automatically calculated status:
- ✅ On Track  
- ⚠️ Almost Due  
- ❌ Overdue  

### 📚 Interaction History
Keep a full record of all past interactions to understand communication patterns.

### 📱 Responsive Design
Optimized for mobile, tablet, and desktop devices.
